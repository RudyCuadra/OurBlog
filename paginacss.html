CSS

CSS es un lenguaje de hojas de estilo utilizado para describir la forma en que un documento HTML o XML debe buscar a un usuario. CSS es donde usted especifica el color, tamaño, espaciado, fuente y otros aspectos visuales del contenido que crea en su documento de lenguaje de marcas.

La mayoría de las veces verá CSS utilizado junto con HTML para describir el aspecto y la apariencia de una página web. Puede tener una página web sin CSS, pero sería muy difícil hacer que se vea como lo desea con solo HTML. Es por esto que casi todas las páginas web son una combinación de HTML y CSS.

CSS significa "Cascading Style Sheets". Un lenguaje de hojas de estilo para describir cómo mostrar un documento HTML.

Selectores

Los selectores son indicadores de HTML que funcionan en CSS para cambiar sus propiedades y estilos, con el fin de mejorar su apariencia.

Algunos ejemplos pueden ser: "p" , se usa para darle un estilo a todos los párrafos de la página "a" , se usa para darle un estilo a una etiqueta especifica

"*" , se usa para cambiar propiedades de todo los elementos en la página "body" , se usa para cambiar el estilo de todos los elementos en el cuerpo 

						
							p {
							/*estilo a párrafos*/
							}

							a {
							/*estilo a etiqueta específica*/
							}

							* {
							/*estilo a todos los elementos de la página*/
							}

							body {
							/*estilo para los elementos del cuerpo*/
							}

Estilo heredado

Las etiquetas que contienen otras etiquetas son padres, y las etiquetas dentro de ellas son sus hijos en la estructura del script CSS, y por ende heredan el estilo y propiedades. A esto se le conoce como estilo heredado.

También se puede combinar múltiples selectores, la selección de estos selectores en igual o diferentes puntos jerárquicos y visuales de la página proporciona más dinamismo al momento de editar el estilo de los elementos y facilidad para modificar la apariencia de la página.

Propiedades

Las propiedades es el nombre de los atributos que cada selector posee. Cada propiedad tiene una colección de valores posibles. Algunos son valores de texto, otros son numéricos y de valor porcentual. 

					
						body {
							background-color: red; /* key word */
							color: #FFFFFF; /* HEX code */
							width: 80%; /* percentage */
							font-size: 15pt; /* numerical value */
						}

Propiedades de color

  • color

Esta propiedad establece el color de primer plano del contenido de texto de un elemento. 

					
						p {
						color:red;
						}
  • background-color

Esta propiedad establece el color de fondo de un elemento. 

					
						a {
						background-color:red;
						}
  • color as a value

Estas propiedades de color toman un color como su valor, y hay tres formas diferentes de definir ese color: palabra clave, un código HEX o un valor rgb. 

					
						/*palabra clave*/ 
						body {
						background-color: teal; 
						}

						/*código HEX*/
						body {
						background-color: #00CC00; /* green */
						}

						/*decimal*/
						body {
						background-color: rgb(0,204,0); /*same green as above*/
						}

Propiedades de fuente

  • Familia tipográfica

Esta propiedad establece la fuente de la fuente. Por lo general, distintos navegadores cargan distintas fuentes de fuentes para cargar sus búsquedas. Por eso se necesita que al momento de hacer un sitio web único, establecer una familia tipográfica para que ésta se mire igual en los diferentes navegadores. 

					
						p {
						font-family: tims, cursive, serif;
						}
  • Tamaño de fuente

El tamaño de fuente establece la escala general de su texto.

					
						h1 {
						font-size: 10em;
						}
  • Peso de la fuente

El peso de una fuente es el grosor de las letras.

					
						p {
						font-weight: bold;
						}
  • Estilo de fuente

La propiedad de estilo de fuente ajusta el ángulo de las letras en relación con el plano horizontal. 

					
						p {
						font-style: italic;
						}
  • Decoración de texto

La decoración de texto agrega una línea a través de tu texto.

					
						p {
						text-decoration: underline;
						}

Propiedades de espaciado

  • Padding

Padding o relleno, es el espacio en blanco que se encuentra más cerca de un elemento HTML.

  • Border

Border o borde, es el área fuera del relleno de un elemento HTML.

  • Margin

Margin o margen de un elemento HTML, es el espacio en blanco que se encuentra fuera del borde.
